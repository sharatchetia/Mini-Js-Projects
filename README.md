# Mini JS Projects

This repository is a collection of small JavaScript projects and practice files built while learning and improving core JavaScript skills.

The goal of this repo is simple:

- practice JavaScript regularly through hands-on mini projects
- improve understanding of DOM manipulation, logic building, and basic UI structure
- keep track of progress over time by adding more small projects

## Current Contents

### 1. JavaScript Practice Files

The root-level files are used for experimenting with JavaScript concepts such as:

- variables and data types
- objects
- factory and constructor functions
- loops
- basic language behavior

Files:

- `index.html`
- `index.js`

### 2. To-Do List Mini Project

A simple to-do list interface created to practice basic frontend structure and styling.

Files:

- `to_Do_List_Mini_Project/index.html`
- `to_Do_List_Mini_Project/style.css`
- `to_Do_List_Mini_Project/scripts.js`

## Purpose

This is a learning repository, not a polished production project. It is meant to document practice, experiments, and gradual improvement through small builds.

More mini projects will be added as learning continues.
